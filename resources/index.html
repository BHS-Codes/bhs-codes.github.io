Resources

hello!

This page is where you'll find any websites, guides, or tutorials we bring up during club. You'll also find a whole lot of learning resources as well as some apps/websites we use fairly frequently.

learning

Scrimba
  • Interactive, beginner-friendly video lessons
  • A handful of free courses for some fundamentals
    • Github Education users get a month of Scrimba Pro + Pro courses for free
The Odin Project
  • Web development (HTML/CSS, JS) + some other good-to-know things (such as Git)
  • Reading-heavy, but effective
  • Entirely free and online!
  • I recommend using Github Codespaces for this
freecodecamp.org
  • Interactive lessons; learn by doing
  • IMO doesn't explain concepts very well but still a solid resource
MDN Web Docs
  • Great web development (+ JS) cirriculum!
    • Also has a focus on web accessibility
  • Jersev used this one and can confirm it's great
  • Reading-heavy but you learn a whole lot.

programs/organizations

Hack Club
  • Online community of a bunch of cool people who like to code things
    • All skill levels!
  • Runs You Ship We Ship events (YSWS)
    • If you code cool things, they'll ship you some cool stuff (stickers, raspberry pis, flipper zeroes, etc)
  • Join the Slack community! Jersev's there sometimes.
Github Education
  • Verify that you are a student -> get Github Pro plus other services for free!
    • GitHub Codespaces Hours
    • One month of Scrimba Pro Courses
    • Discounts on some Adafruit + Arduino products
  • Really simple sign up process, you can send a picture of your class schedule/student ID/transcripts

(fairly) frequent usage

Github
  • We use Github to collaborate on projects (such as this website, hosted on Github! [don't look at the source code it's not good])
  • We have a BHSC organization page that we will get you guys added to later in the year.
  • Don't worry too much if you have no idea what you're looking at when you open Github, we'll walk through it once we start using it.
    • (though some light reading never hurts.)>
sprig
  • a mini game-making website similar to scratch
  • if you complete a project in sprig, you can get a sprig machine sent to you!
  • learn more on the sprig website.
teachable machine
  • teach an AI to differentiate between things
  • use your webcam/microphone to input training data to the AI
  • let the AI train a little bit...
  • ...and you'll have an AI that can sort/differentiate things in real-time!
  • (we played around with this on 2025-09-18.)
