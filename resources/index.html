Resources

learning

Scrimba
Uses (actually) interactive video lessons to teach stuff! Very good for beginners and has a couple of free courses. Github Education (see below) users can get free access to pro courses for one month.
The Odin Project
A reading-heavy but effective course for web development. Entirely free and online!
freecodecamp.org
Interactive lessons that let you learn by doing. Not very in-depth but still a solid resource.
MDN Web Docs
Really good resource with a great curriculum! I learned some HTML/CSS through here, it's reading-heavy but you learn a whole lot.

programs/organizations

Hack Club
An online community of a bunch of cool people who like to code things. All skill levels, and they run events where you can get cool stuff if you code stuff (You Ship We Ship [YSWS]). Join the slack! Jersev's there sometimes.
Github Education
Verifying that you're a student give you access to Github Pro, which gives you access to a bunch of Github Codespaces hours. On top of this, you gain access to a bunch of services (such as Scrimba!) and courses for free. HIGHLY recommend signing up, the process is super simple. (but let me know if you need help it was a bit of a search...)
